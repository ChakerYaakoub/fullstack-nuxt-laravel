API: `POST http://127.0.0.1:8000/api/auth/login`

## Run

Terminal 1 — Laravel:
```bash
cd backend
php artisan serve
```

Terminal 2 — Admin:
```bash
cd admin
npm run dev
```

Open http://localhost:5173
